$\huge\color{cadetblue}{\text{Problem 1}}$

<br/>

For each of the following annotations determine which choice fits on the empty line (.....). The variables x, y, and z are of type int. Note that A and B (capital letters!) are specification constants (so not program variables).

---------------

$\Large{\color{darkkhaki}\text{Prob 1.1}}$

```java
// true 
.....
// x + y = 42 
```

${\color{peru}(a)} \quad x := 6*y; \space y := 3;$  
${\color{peru}(b)} \quad x := x - y; \space y := 21;$  
${\color{peru}(c)} \quad y := 14; \space x := 2*y;$  

<br/>

---------------

$\Large{\color{darkkhaki}\text{Prob 1.2}}$

```java
// x = A ∧ y = B 
.....
// x = B ∧ y = A 
```

${\color{peru}(a)} \quad x := y - 2*x; \space y := y - x;$  
$\qquad \space x := x + y; \space y := y/2;$  
${\color{peru}(b)} \quad x := y - 2*x; \space y := y - x/2;$  
$\qquad \space x := x + 2*y;$  
${\color{peru}(c)} \quad y := x + y; \space x := y - x;$  
$\qquad \space y := y + x$  

<br/>

---------------

$\Large{\color{darkkhaki}\text{Prob 1.3}}$

```java
// x * y = A * B 
.....
// x * y = A * B 
```

${\color{peru}(a)} \quad x := x + 1; \space y := y - 1;$  
${\color{peru}(b)} \quad  x := 1; \space y := x*y - y;$  
$\qquad \space x := x + 1;$  
${\color{peru}(c)} \quad x := x + 1; \space y := x*y - y;$  
$\qquad \space x := 1;$  

<br/>

---------------

$\Large{\color{darkkhaki}\text{Prob 1.4}}$

```java
// ..... 
x := 2 * x; 
y := 3 * x - 20;
// x < y
```

${\color{peru}(a)} \quad x \geq 5$  
${\color{peru}(b)} \quad x > 5$  
${\color{peru}(c)} \quad x > -5$  

<br/>

$\Large{\color{darkkhaki}\text{Prob 1.5}}$

```java

```

${\color{peru}(a)} \quad x > 25$  
${\color{peru}(b)} \quad x > 5$

${\color{peru}(b)} \quad x < 10$
