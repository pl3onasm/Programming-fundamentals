$\huge\color{cadetblue}{\text{Summing with step k}}$

<br/>

The input for this exercise is a line with three numbers $n$,$m$, and $k$. The output should be the sum of the numbers from $n$ until $m$ (both inclusive) in steps of size $k$. For example, when the input is $5 \space 12 \space 3$, the sum of the numbers $5 + 8 + 11 = 24$. The time limit for this calculation is 1 second.

<br/>

$\Large\color{darkseagreen}{\text{Examples}}$

```text
input:
  5 12 3
output:
  24

input:
  1234567 9876543 2
output:
  24005492043895

input:
  12345678 98765432 5
output:
  960219455829903
```
